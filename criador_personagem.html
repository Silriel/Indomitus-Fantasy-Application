Criador automatico de personagem

Preencha os campos principais abaixo e gere uma ficha base automaticamente. Depois voce pode continuar os ajustes no criador pessoal.

Preencha, gere e ajuste depois
Passo 1 Escolha os dados principais

Defina o nivel, a classe e a raca que vao servir de base para a ficha.

Passo 2 Revise a subclasse

Se o nivel permitir, escolha uma subclasse para completar a base do personagem.

Passo 3 Gere a ficha

Use o resultado como base rapida ou leve a ficha para o modo manual depois.

Dados iniciais

Nivel, classe e raca

Esses campos definem os atributos, os recursos e as passivas iniciais da ficha gerada.

Use o nivel base para calcular atributos, HP, MP, SP e progresso.
A classe define a base de HP, MP, SP e a prioridade de atributos.
A subclasse aparece quando o nivel for 15 ou maior e a classe tiver essa opcao.
O bonus racial base entra no nivel 1 e se repete nos niveis 21, 41, 61 e 81.
Use este campo quando a raca tiver estagio ou variante propria.
Escolha 2 ou mais racas diferentes. Primeiro a ficha calcula a media dos bonus raciais, arredonda para baixo e depois repete esse bonus nos niveis 21, 41, 61 e 81. As passivas raciais sao opcionais e voce pode marcar quantas quiser entre as racas escolhidas.
Escolha uma raca para ver passiva, vantagens e desvantagens.
